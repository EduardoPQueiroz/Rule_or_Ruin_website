O que é Rule or ruin?

Rule or ruin é um jogo criado pelos alunos da Etec Horácio como projeto para a semana do industrial.

O que é a semana do industrial ?

A semana do industrial é uma semana dedicada aos alunos desenvolverem projetos e apresentarem para pessoas de fora, para que elas tenham conhecimento do que está sendo trabalhado dentro da escola e assim elas criem interesses.

Como o jogo funciona ?

Trata-se de um jogo de escolhas que se passa na Alemanhã durante diferentes anos

No jogo, você é um rei responsável por fazer escolhas que alteram os seus diferentes status dentro do jogo

Como foi a criação do game ?

O game foi desenvolvido utilizando diferentes tecnologias que aprendemos durante o curso.

Algumas das tecnologias usadas

Java

O java é uma linguagem de programação versátil, reconhecida pela portabilidade e escalabilidade em várias plataformas.

SQL Server

O SQL é uma linguagem padrão para gerenciar bancos de dados relacionais eficiente para consultar e manipular dados.

.Net-Maui

O .net maui é uma tecnologia que permite o desenvolvimento de aplicativos para múltiplas plataformas usando um único código.

Java Script

O javascript é uma linguagem essencial para a web, usada para criar interfaces dinâmicas e interativas no navegador.

Html

O HTML é uma linguagem de marcação que estrutura o conteúdo e os elementos visuais de páginas web.

CSS

O CSS é uma linguagem que define a aparência de páginas web, controlando layout, cores e fontes.
